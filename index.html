什么是记忆球

Obsidian 情绪可视化插件

🧠 情绪标签 → 视觉记忆球

在日记中用 #情绪/开心 等标签记录心情，插件自动为每种情绪生成一个发光的彩色球体。支持混合球（多情绪合成）和核心记忆（脉冲光晕）两种特殊形态。

🫧 管道式物理交互

球体在圆角管道中排列。拖拽产生弹性碰撞，释放后自然衰减回原位。模拟一维弹性碰撞（动量守恒 + 恢复系数控制能量损失）。

📊 数据驱动

统计面板追踪情绪频率、分布和趋势，含核心记忆占比分析。

多排管道

球数超过单排容量时按插件逻辑自动分行，每根管道独立响应拖拽与碰撞

🔄 自动换行

超出单排容量后，自动创建下一根管道，每根独立运作，物理引擎确保排列整齐。

Markdown 嵌入

在任何笔记中用代码块插入记忆球管道

```memory-orbs
date: 2026-05-02
mode: week
theme: cosmos
minimal: true
```
→
渲染为 Obsidian 内的记忆球管道

插件实际支持 date / mode / theme / minimal，也可用 start + end 指定自定义范围。
