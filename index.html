Markdown 嵌入

在 Obsidian 笔记里直接嵌入记忆球，无需离开编辑页

```memory-orbs
date: 2026-04-29
mode: day
theme: glass
minimal: true
```

✦ 支持 date · mode · theme · start/end

✦ 按住 Cmd + 点击球体 → 打开对应日记

✦ 代码生成器：Cmd+P → "代码生成器"
