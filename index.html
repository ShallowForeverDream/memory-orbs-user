什么是记忆球

Obsidian 情绪可视化插件

🧠 情绪标签 → 视觉记忆球

在日记中用 #情绪/开心 这样的标签记录心情，插件自动为每种情绪生成一个发光的彩色球体。灵感来自 Pixar《头脑特工队》。

🫧 管道式交互

球体在圆角管道中排列、滚动、碰撞，模拟真实的物理效果。拖拽时产生弹性摆动，释放后自动衰减回原位。

📊 数据驱动

统计面板追踪情绪频率、分布和趋势，让你直观了解自己的情绪变化。

添加 Memory Orbs

BRAT → Add Beta Plugin → 输入 https://gitee.com/housiyuan123/memory-orbs-user

3

获取激活码

点击上方「购买」按钮，扫码获取激活码

4

开始记录心情！

在日记中使用 #情绪/开心 等标签，打开管道视图查看你的记忆球
